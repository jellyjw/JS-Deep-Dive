## 연산자

연산자는 하나 이상의 표현식을 대상으로 산술, 할당, 비교, 논리, 타입, 지수 연산 등을 수행해 하나의 값을 만든다. 이때 연산의 대상을 피연산자라고 하며, 피연산자는 값으로 평가될 수 있는 표현식이어야 한다.

자바스크립트는 다양한 연산자를 제공한다.

### 1) 산술 연산자

피연산자를 대상으로 수학적 계산을 수행해 새로운 숫자 값을 만든다. 만약 산술 연산이 불가능할 경우, `NaN` 을 반환한다.

```jsx
let x = 'hello'
x + 5 = NaN
```

### 1.1) 이항 산술 연산자

2개의 피연산자를 산술 연산하여 숫자 값을 만든다. 이때 모든 이항 산술 연산자는 **값을 변경하는 부수 효과가 없다.**

즉, 어떤 연산을 수행하던간에 피연산자의 값이 바뀌는 경우 없이 언제나 새로운 값을 만든다.

```jsx
5 + 2; // 7
5 % 1; // 1
```

### 1.2) 단항 산술 연산자

1개의 피연산자를 산술 연산하여 숫자 값을 만든다.

| 연산자 | 설명                                               | 예제             | 부수 효과 |
| ------ | -------------------------------------------------- | ---------------- | --------- |
| +      | 어떠한 효과도 없다. 음수를 양수로 반전하지 않는다. | let x = +5;      | X         |
| -      | 양수를 음수로, 음수를 양수로 반전한 값을 반환한다. | let y = -5;      | X         |
| ++     | 증가 (1을 더함)                                    | let a = 10; a++; | O         |
| --     | 감소 (1을 뺌)                                      | let b = 10; b--; | O         |

이항 산술 연산자와 달리 증가, 감소 연산자(`++` , `--` ) 는 **피연산자의 값을 변경하는 부수 효과가 있다** 는 것을 주의해야 한다. 값을 변경하는 암묵적 할당이 이뤄지기 때문이다.

단항 연산자(`+` , `-` )는 부수효과가 없으며 숫자 타입이 아닌 피연산자에 + 단항 연산자를 사용할 경우 피연산자를 숫자 타입으로 변환하여 반환한다. 이때 새로 값이 생성되기 때문에 부수효과가 없다.

```jsx
let x = "1";

console.log(+x); // 1
console.log(x); // '1' 부수효과 없음

x = true;
console.log(+x); // 1

x = false;
console.log(+x); // 0
```

위 예제처럼 `boolean` 값을 숫자로 타입 변환하여 반환한다. 피연산자를 직접 변경하지는 않는다.

### 1.3) 문자열 연결 연산자

`+` 연산자는 **피연산자 중 하나 이상이 문자열인 경우 문자열 연결 연산자로 동작**한다. 그 외의 경우는 산술 연산자로 동작한다.

```jsx
let x = '1'

1 + '1' = '11'
1 + '2' = '12'

1 + true // 2

1 + false // 1

1 + null // 1, null이 0으로 타입 변환

+undefined // NaN
1 + undefined // NaN
```

개발자의 의도와는 상관없이, JS 엔진에 의해 암묵적으로 타입이 자동 변환되기도 한다는 것을 주목하자.

이를 **암묵적 타입 변환** 또는 **타입 강제 변환** 이라고 한다.

### 2) 할당 연산자

할당 연산자는 우항에 있는 피연산자의 평가 결과를 좌항에 있는 변수에 할당한다. 좌항의 변수에 값을 할당하므로 **변수의 값이 변하는 부수효과가 있다.**

| 연산자 | 설명                 | 예제       | 부수효과 |
| ------ | -------------------- | ---------- | -------- |
| =      | 대입 (할당)          | x = 5;     | O        |
| +=     | 더하고 할당          | x += 3;    | O        |
| -=     | 빼고 할당            | x -= 2;    | O        |
| \*=    | 곱하고 할당          | x \*= 4;   | O        |
| /=     | 나누고 할당          | x /= 2;    | O        |
| %=     | 나머지를 구하고 할당 | x %= 3;    | O        |
| \*\*=  | 거듭제곱하고 할당    | x \*\*= 2; | O        |

할당문은 **값으로 평가되는 표현식인 문**으로서, 할당된 **값으로 평가** 된다. 따라서 할당문을 다른 변수에 할당할수도 있다.

```jsx
var a, b, c;
a = b = c = 0;

console.log(a, b, c); // 0 0 0
```

### 3) 비교 연산자 및 대소 관계 비교 연산자

좌항과 우항의 피연산자를 비교한 다음 `boolean` 값으로 반환한다. 주로 for문이나 if문 과 같은 제어문의 조건식에서 사용된다.

| 연산자 | 설명             | 예제    | 부수효과 |
| ------ | ---------------- | ------- | -------- |
| ==     | 동등 비교        | x == y  | X        |
| ===    | 일치 비교        | x === y | X        |
| !=     | 부정 동등 비교   | x != y  | X        |
| !==    | 부정 일치 비교   | x !== y | X        |
| <      | 작음 비교        | x < y   | X        |
| >      | 큼 비교          | x > y   | X        |
| <=     | 작거나 같음 비교 | x <= y  | X        |
| >=     | 크거나 같음 비교 | x >= y  | X        |

- 동등 비교 연산자 : `==` 연산자는 **좌항과 우항의 피연산자를 비교할 때 암묵적 타입 변환을 통해 타입을 일치시킨 후 같은 값인지 비교** 한다. 따라서 타입이 다르더라도 타입 변환 후 같은 값이라면 `true` 를 반환한다.

```jsx
5 == "5"; // true
5 == 5; // true
```

동등 비교 연산자는 결과를 예측하기 어렵고 실수하기 쉽다. 따라서 안티 패턴으로 일치 비교 연산자를 사용하는게 좋다.

- 일치 비교 연산자 : `===` 연산자는 **좌항과 우항의 피연산자가 타입도 같고 값도 같은 경우에 한하여 `true` 를 반환**한다. 다시 말해 타입 변환을 하지 않고 비교하기 때문에 예측이 쉽다.

```jsx
5 === 5; // true
5 === "5"; // false
```

일치 비교 연산자에서 주의할 점은, `NaN` 은 자신과 일치하지 않는 **유일한** 값이라는 것이다.

```jsx
NaN === NaN; // false

// 빌트인 함수 isNaN 을 사용하자.

isNaN(NaN); // true
isNaN(10); // false
```

숫자 0 또한, JS에서는 양의 0과 음의 0을 비교해도 `true` 를 반환한다.

### 4) 삼항 조건 연산자

조건식의 평가 결과에 따라 반환할 값을 결정한다. JS의 유일한 삼항 연산자로 부수효과는 없다.

```jsx
조건식 ? 조건식이 true 일때 반환할 값 : 조건식이 false 일때 반환할 값
```

삼항 조건 연산자의 첫번째 피연산자는 조건식이므로 조건문과 같다고 할 수 있다. 따라서 `if ... else` 문을 사용해도 유사하게 처리할수 있다.

### if …else 문과 삼항연산자의 차이

`if...else` 문은 표현식이 아닌 문이기 때문에 값처럼 쓸 수 없다. 하지만 삼항연산자는 값처럼 사용할수 있다. 따라서 매우 유용하게 사용된다.

```jsx
let x = 10;
result = x % 2 ? "홀수" : "짝수";
console.log(result); // 짝수
```

### 그 외 연산자

그 외에도 논리 연산자, 쉼표 연산자, 그룹 연산자, typeof 연산자, ES7에서 도입된 지수 연산자 등 수많은 연산자가 있는데, 연산자를 사용할 경우에는 항상 부수효과에 주의해서 사용해야 한다.
